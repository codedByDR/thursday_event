3-Tier AI Integration Architecture

Layer 1: User Experience

User Interface (UI)

Employee submits requests & checks status.

Layer 2: Protected API & Logic Core

API Gateway

Handles requests, routing, & PII sanitization.

AI Agents

Drafts requests, parses logic, & flags HR bias.

Layer 3: Data & External Integration

Databases

Vector DB (Policies) & HRIS (Internal Data).

External Systems

Mail, Telegram & Childcare booking APIs.

Data Isolation Enforced: The User Interface (Layer 1) and Data/External Systems (Layer 3) can only communicate through the secure API Gateway (Layer 2). The AI Agents operate exclusively behind the API to ensure strict PII sanitization and policy enforcement.
